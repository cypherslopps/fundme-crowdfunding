# FundME Crowdfunding DAPP

### Stack
- Redux Toolkit
- GSAP
- Lodash for debouncing
- Ethers.js
- Shadncn
- Rainbowkit for Wallet Connect
- Tailwindcss
- Filepond for Image Upload
- UUID for generating random generic IDs
- Firebase for storing images
